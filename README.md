## インストール
```sh
$ npm i -g @kobalab/liulian
```

## 起動
```sh
$ liulian ~/Documents/LiuLian
```

## 使用方法

### liulian [--port=*port*] [--mount=*url*] *workdir*
<dl>
<dt>--port, -p</dt>
  <dd>サーバを起動するポート番号 (デフォルト値は 3571)</dd>
<dt>--mount, -m</dt>
  <dd>リバースプロキシのマウントポイントのURL</dd>
<dt>workdir</dt>
  <dd>コンテンツや設定ファイルを配置するディレクトリ</dd>
</dl>
